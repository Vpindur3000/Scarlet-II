# Changelog

## v1.0.0 — Modern B* source release

- First public source release of the current Modern B* branch.
- Added complete GPLv3 licensing and explicit third-party attribution.
- Removed all neural-network weights from the repository and release archive.
- Removed prebuilt binaries, generated build directories, caches, and rollback snapshots.
- Documented that non-terminal search intervals are heuristic rather than formal minimax proofs.
- Added external-network and release-packaging guidance.

Historical development notes remain in `docs/` and may refer to earlier internal
version numbers.
